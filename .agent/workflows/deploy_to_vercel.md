---
description: Deploy the application to Vercel
---

1. Login to Vercel
   - Run the following command to log in to your Vercel account.
   - `npx vercel login`

2. Deploy to Preview (Optional)
   - Run this command to deploy a preview version of your app.
   - `npx vercel`

3. Deploy to Production
   - Run this command to deploy your application to production.
   - `npx vercel --prod`
